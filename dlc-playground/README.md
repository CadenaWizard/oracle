# DLC Playground

Playground for Discrete Log Contracts (DLC) and Adaptor Signatures, written in Rust, focusing on the cryptography.

Reading: [main.rs](src/main.rs)

Running: simply

```shell
cargo run
```

Prerequisite: Rust

Credits: Partly insipred by [rust-dlc](https://github.com/p2pderivatives/rust-dlc)
